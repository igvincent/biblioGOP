# BiblioGOP

Library management

- [Firebase](https://www.firebase.com/)
- [Angular](https://angularjs.org/)
- [AngularFire](https://www.firebase.com/docs/web/libraries/angular/index.html)
- [Compass](http://compass-style.org/) / [Sass](http://sass-lang.com/)
- [Bower](http://bower.io/)

##Install

- `bower instal`
- `grunt compass`
- `grunt server`
